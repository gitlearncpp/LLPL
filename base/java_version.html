Java 1.0 (1996)

Pierwsza wersja Javy: Pierwsze wydanie języka Java przez Sun Microsystems. Zawierało podstawowe elementy języka, takie jak model obiektowy, automatyczne zarządzanie pamięcią (garbage collection) oraz standardowe biblioteki API, które wspierały GUI (AWT), obsługę sieci, wielowątkowość i inne podstawowe funkcje.

-----------------------------------------------------------------------------
Java 1.1 (1997)

Inner Classes: Wprowadzenie klas wewnętrznych.
Reflection: Możliwość inspekcji i wywoływania metod i pól klas w czasie wykonania.
JDBC: Pierwsza wersja Java Database Connectivity do komunikacji z bazami danych.
RMI: Zdalne wywoływanie metod (Remote Method Invocation) pozwalające na wywoływanie metod na obiektach znajdujących się na innych maszynach w sieci.

-----------------------------------------------------------------------------
Java 1.2 (1998) – Java 2

Swing: Wprowadzenie nowoczesnej biblioteki GUI Swing, która była bardziej zaawansowana i elastyczna niż AWT.
JIT Compiler: Poprawa wydajności dzięki kompilatorowi Just-In-Time (JIT).
Collections Framework: Nowa struktura kolekcji (List, Set, Map).
Java Plug-in: Umożliwił uruchamianie apletów Javy w przeglądarkach.

-----------------------------------------------------------------------------
Java 1.3 (2000)

HotSpot JVM: Nowa maszyna wirtualna z dynamiczną kompilacją i optymalizacją kodu.
RMI-IIOP: Integracja RMI z protokołem IIOP, umożliwiająca komunikację z aplikacjami CORBA.
Synthetic Proxy Classes: Nowa funkcjonalność dynamicznego tworzenia klas proxy.

-----------------------------------------------------------------------------
Java 1.4 (2002)

assert: Nowe słowo kluczowe do obsługi asercji.
NIO: New I/O API, które wprowadziło bufory i kanały, oferując bardziej efektywne zarządzanie danymi we/wy.
Logging API: Wprowadzenie standardowego API do logowania.
XML Parsing: Wbudowane wsparcie dla przetwarzania XML (SAX, DOM).

-----------------------------------------------------------------------------
Java 5 (2004) – znana również jako 1.5

Generics: Umożliwiły tworzenie klas i metod z typami generycznymi, co zwiększyło bezpieczeństwo typów w czasie kompilacji.
Enhanced for Loop: Wprowadzenie udoskonalonej pętli for-each.
Autoboxing/Unboxing: Automatyczne konwersje między typami prostymi (prymitywnymi) a ich odpowiednikami klasowymi.
Enums: Wprowadzenie wyliczeń jako specjalnego typu danych.
Annotations: Dodanie adnotacji do Javy, umożliwiających metadane w kodzie źródłowym.
Concurrency API: Nowe klasy wspomagające programowanie wielowątkowe.

-----------------------------------------------------------------------------
Java 6 (2006)

Scripting API: Integracja z innymi językami skryptowymi dzięki API.
Compiler API: Udostępnienie API dla kompilatora.
Pluggable Annotations: Wsparcie dla adnotacji procesowanych w czasie kompilacji.
JDBC 4.0: Rozbudowa API JDBC, w tym uproszczona obsługa zasobów.

-----------------------------------------------------------------------------
Java 7 (2011)

Project Coin: Wprowadzenie kilku małych, ale użytecznych zmian w składni, takich jak: diamond operator (<>), multi-catch, try-with-resources.
NIO.2: Rozbudowa API New I/O, w tym wsparcie dla pracy z systemem plików i asynchroniczne operacje I/O.
Fork/Join Framework: Nowa struktura do wykonywania zadań równoległych, optymalizująca prace wielowątkowe.
InvokeDynamic: Usprawnienia JVM do lepszej współpracy z językami dynamicznymi.

-----------------------------------------------------------------------------
Java 8 (2014)

Lambdy: Wprowadzenie wyrażeń lambda, umożliwiających tworzenie funkcji anonimowych.
Stream API: Nowe API do przetwarzania danych w sposób funkcyjny i równoległy.
Optional: Nowa klasa, która umożliwia reprezentowanie wartości opcjonalnych, eliminując częste problemy z null.
Date and Time API: Nowe API do obsługi dat i czasu, oparte na bibliotece Joda-Time.
Default Methods: Możliwość definiowania domyślnych metod w interfejsach.

-----------------------------------------------------------------------------
Java 9 (2017)
Jigsaw (Modularity): System modułów, umożliwiający dzielenie aplikacji na mniejsze moduły, co zwiększa bezpieczeństwo i poprawia zarządzanie zależnościami.
JShell: Interaktywny interpreter, który umożliwia eksperymentowanie z kodem Java.
Collection Factory Methods: Nowe metody do tworzenia niezmiennych kolekcji w prostszy sposób.
Reactive Streams API: Wsparcie dla programowania reaktywnego.

-----------------------------------------------------------------------------
Java 10 (2018)
Local-Variable Type Inference: Wprowadzenie słowa kluczowego var umożliwiającego inferencję typu zmiennej lokalnej.
Application Class-Data Sharing: Optymalizacje związane z ładowaniem klas, co przyspiesza start aplikacji.
Garbage Collector Improvements: Poprawki w wydajności GC, w tym eksperymentalny GC „ZGC”.

-----------------------------------------------------------------------------
Java 11 (2018)

LTS (Long Term Support): Pierwsza wersja LTS od Java 8.
ZGC (Z Garbage Collector): Nowy GC o niskich opóźnieniach.
HTTP Client API: Nowe API do obsługi żądań HTTP/2.
Flight Recorder: Narzędzie do rejestrowania i analizy danych diagnostycznych JVM.
Removal of Deprecated Features: Usunięcie starych funkcji, takich jak Applets i JNLP.

-----------------------------------------------------------------------------
Java 12 (2019)

Switch Expressions (Preview): Ulepszony switch jako wyrażenie z możliwością zwracania wartości.
JVM Constants API: Nowe API do pracy z constantami w JVM.
Improved Garbage Collection: Dalsze usprawnienia w ZGC oraz nowa implementacja Shenandoah GC.

-----------------------------------------------------------------------------
Java 13 (2019)

Text Blocks (Preview): Umożliwienie łatwiejszego definiowania wielowierszowych tekstów.
Reimplementacja Socket API: Przebudowa klasy Socket i powiązanych klas.
Dynamic CDS Archives: Ulepszenia w tworzeniu archiwów klas dla szybszego uruchamiania aplikacji.

-----------------------------------------------------------------------------
Java 14 (2020)

Switch Expressions (Final): Ukończenie prac nad ulepszonym switch.
Pattern Matching for instanceof (Preview): Ulepszenia w operacjach typu instanceof, eliminujące potrzebę rzutowania.
Records (Preview): Nowa składnia ułatwiająca tworzenie klas przechowujących dane.

-----------------------------------------------------------------------------
Java 15 (2020)
Text Blocks (Final): Finalizacja funkcji bloków tekstowych.
Sealed Classes (Preview): Nowa funkcjonalność pozwalająca na tworzenie klas ograniczonych do określonych podklas.
Hidden Classes: Możliwość tworzenia klas, które nie są bezpośrednio widoczne dla kodu aplikacji.

-----------------------------------------------------------------------------
Java 16 (2021)
Records (Final): Ukończenie prac nad rekordami.
Pattern Matching for instanceof (Final): Ukończenie funkcjonalności dopasowywania wzorców dla instanceof.
Vector API (Incubator): Nowe API umożliwiające pracę z wektorami SIMD.

-----------------------------------------------------------------------------
Java 17 (2021)

LTS: Kolejna wersja LTS.
Sealed Classes (Final): Finalizacja klas zamkniętych.
Pattern Matching for switch (Preview): Umożliwienie dopasowania wzorców w instrukcjach switch.
Foreign Function & Memory API (Incubator): Nowe API do współpracy z kodem natywnym i zarządzania pamięcią poza stertą.

-----------------------------------------------------------------------------
Java 18 (2022)

Simple Web Server: Dodanie prostego serwera HTTP dla celów testowych.
UTF-8 by Default: UTF-8 staje się domyślnym kodowaniem znaków.
Vector API (3rd Incubator): Kolejna wersja API do obsługi wektorów SIMD.

-----------------------------------------------------------------------------
Java 19 (2022)
Virtual Threads (Preview): Eksperymentalne wsparcie dla wirtualnych wątków, które pozwalają na tworzenie lekkich wątków na JVM.
Structured Concurrency (Incubator): API wspierające lepsze zarządzanie współbieżnymi operacjami.

-----------------------------------------------------------------------------
Java 20 (2023)

Pattern Matching for switch (Second Preview): Dalszy rozwój dopasowywania wzorców dla switch.
Scoped Values (Incubator): Nowe API pozwalające na bezpieczne przekazywanie danych między różnymi wątkami.
Foreign Function & Memory API (Preview): Kontynuacja rozwoju API do współpracy z kodem natywnym.

-----------------------------------------------------------------------------
Java 21 (2023)

LTS: Kolejna wersja LTS.
Virtual Threads (Final): Finalizacja wsparcia dla wirtualnych wątków.
Sequenced Collections: Wprowadzenie nowego typu kolekcji z zachowaną kolejnością elementów.
Pattern Matching for switch (Final): Finalizacja dopasowywania wzorców dla switch.
String Templates (Preview): Wprowadzenie nowej funkcji umożliwiającej wstawianie zmiennych bezpośrednio do łańcuchów znaków.
Każda z tych wersji wprowadziła unikalne zmiany, które kształtowały i rozwijały język Java, dostosowując go do współczesnych potrzeb i trendów w programowaniu.
